The short version

I'm a software engineer who has spent the last few years focused on mobile development, primarily with Flutter and Dart, alongside React and Bootstrap on the web side. Most of my work has been inside Nigerian fintech and startup teams, building apps that move real money and real data for real users.

My day-to-day covers UI implementation, state management, API integration, and wiring up Firebase as a backend — plus the less glamorous parts: local storage, security, bug fixing, and keeping an app's full lifecycle (concept → design → test → release → support) moving.

I care about clean architecture and design implementation that holds up once an app is actually in production, not just in a demo.
